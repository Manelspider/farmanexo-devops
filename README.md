# FarmaNexo DevOps

## Descripción

Repositorio encargado de la infraestructura de FarmaNexo. Contiene la configuración de Docker Compose, redes, volúmenes, Traefik y otras herramientas necesarias para levantar todo el stack de manera coordinada y reproducible.

## Características principales

- Definición de contenedores para API, Panel, App y base de datos.
- Configuración de Traefik para enrutamiento de subdominios.
- Red compartida entre contenedores para comunicación interna.
- Facilita despliegue local y pruebas de integración.
